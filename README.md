 # Quant Risk Lab

A full-stack quantitative finance project.

Features
- Portfolio Risk Analysis
- Value at Risk (VaR)
- Conditional VaR
- Portfolio Optimization
- Backtesting engine
- React dashboard

Tech Stack

Backend
- Python
- FastAPI
- Numpy
- Pandas

Frontend
- React
- Vite
- Recharts

Infrastructure
- Docker
- GitHub Actions
